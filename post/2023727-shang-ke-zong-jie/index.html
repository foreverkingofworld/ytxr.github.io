2023/7/27上课总结

1. 全局区

外部变量（全局变量）

1.作用域

整个应用程序，如果改变了外部变量，外部变量都会随之改变，可能会影响到整个程序。

2.生命周期

外部变量生命周期与程序生命周期一样。

3. 存储位置

全局区（静态区）

4. 生成时间

程序启动，外部变量生成。

5. 外部变量的定义和使用

(1).外部变量必须在.h文件中声明
extern int a; // 声明不要初始化
(2).外部变量的定义
在程序的任何一个.c文件都可以进行定义，但是必须引入.h文件。
外部变量只能定义一次，定义完后，其他文件可以使用但是不能再次定义。
外部变量定义的位置必须是所有函数的外部。

静态外部变量（静态全局变量）

1.在变量前加上static关键字，在所有函数外定义就是静态外部变量，在函数内定义就是静态内部变量。
2.存储在全局区（静态区）
3.生命周期
与程序生命周期相同
4.作用域
本.c文件内，其他文件不能调用

内部变量（局部变量）

1. 在函数体内定义的所有变量都是局部变量

2. 内部变量存储在栈区

3. 生命周期

函数执行调用这个变量时生成，函数运行结束后，这个内部变量被释放。

4. 作用域

函数体内

块级变量

1. 在函数体内语句块内定义的变量就是块级变量

2. 块级变量存储在栈区

3. 生命周期

在语句块内调用时生成，语句块结束时释放

4. 作用域

语句块内

变量调用原则

1.就近原则，如果函数内有这个变量就调用函数内的而不是外部的。

变量的错误定义

在一个.c文件中不能同时定义同名的外部和静态外部变量
在一个函数中不能同时定义同名的内部和静态内部变量

2. 常量区

常量区的相同字符串只有一个，所有指向同一个字符串的地址是相同的。
生命周期：程序员用指针=这个字符串起开始创建，知道程序结束被释放。
常量区字符串不能被改变。

    char *p = "abc";
	char *p1 = "abc";
	printf("%p\n", p);
	printf("%c\n", p[0]);
	printf("%p\n", p1);
	printf("%c\n", p1[0]);

3. 栈区变量的管理

普通的变量都在栈区，eg:int char double float 等等

4. 代码区

函数不加static就都是外部的，可以被全局调用
加static的函数是静态函数，只能被本.c文件调用，其他文件无法调用
static void a(){}

函数指针

函数也有地址，指向函数的指针就是函数指针

#include <stdio.h>

void a()
{
	printf("A\n");
}


int A(int a, int b)
{
	return a+b;
}

int main()
{
	a();
	printf("%p\n", a);
	
	void (*p)();  // 函数指针的类型就是函数类型加指针加参数列表
	p = a;
	p();
	
	int (*p1)(int a, int b);
	p1 = A;
	int c = p1(100, 200);
	printf("%d\n", c);
}
