2023/7/31上课总结

函数指针与指针函数

函数指针，指向函数的指针，主要应用是函数指针作为参数

指针函数，返回值是指针的叫做指针函数

指针常量与常量指针

指针常量，指针是一个常量

    int a = 10;
	int b = 20;
	int *const p = &a;
	// p = &b; // 指针的指向不能被改变
	*p = 200;
	printf("%d\n", *p);
	printf("%d\n", a);

常量指针，常量是一个指针

    int a = 10;
	const int *p = &a; // *p不能被改变
	int b = 100;
	p = &b;
	printf("%d\n", *p);
